Chat Commands Reference

Send these commands to any chat where the bot is added. Only you and allowed users can use them.

Sending
/send <topic> — send all questions in a topic
/sendq <topic> <number> — send one specific question
/sendall — send all topics
/answer <topic> <number> — send answer to the group
/answerprivate <topic> <number> — send answer privately to you

Topics & Questions
/topics — list all topics
/list <topic> — list all questions in a topic
/addtopic <name> — create a new topic
/addq <topic> | <question> | <answer> — add a question (answer optional)
/delq <topic> <number> — delete a question
/deltopic <topic> — delete a topic

Users
/allow <user_id> — allow a user to use commands
/disallow <user_id> — remove a user's access
/users — list allowed users

Other
/stop — stop current sending session
/help — show command list in chat
⚠️ Setup Required
For commands to work in chat, you need to run the polling bot script on a server or your PC.

Option A — Run locally (PC stays on):
1. Install Node.js
2. Download bot.js below
3. Run: node bot.js

Option B — Free cloud hosting:
Deploy bot.js to Railway.app or Render.com (free tier)
